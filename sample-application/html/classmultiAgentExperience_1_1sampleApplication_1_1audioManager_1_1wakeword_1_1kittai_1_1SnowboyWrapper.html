Detailed Description

The wrapper class for snowboy::SnowboyDetect class.

Since the original API exposes std::string which will be ABI incompatible with GCC 5.1, this class replaces them with const char*.

We keep the actual instantiation in a void* pointer in this header, to prevent from other files to include snowboy-detect.h file.

Thanks to this community for the original idea: https://github.com/Kitt-AI/snowboy/issues/99

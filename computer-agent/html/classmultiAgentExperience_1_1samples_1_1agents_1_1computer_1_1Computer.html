Detailed Description

This class represents an example of a MAX-conformant Agent. It implements an AWS Lex client to parse and respond to simple user requests. It is expected that this Agent will be integrated with an Alexa client via the MAX Library, as some responses from this Agent may refer to Alexa. This class is provided for demonstration purposes only, and is not supported for commercial use.

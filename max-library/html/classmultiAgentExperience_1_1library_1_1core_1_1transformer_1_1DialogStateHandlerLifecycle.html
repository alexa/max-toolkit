Detailed Description

The DialogStateHandlerLifecycle helps manage the lifecycle of any DialogStateHandler. The callbacks are given to take action as the lifecycle changes. Callbacks are used in favor of wrapping the DialogStateHandler itself as there is no interface shared between all state handlers. The lifecycle will prevent the callbacks from being used in the wrong order or the wrong number of times.

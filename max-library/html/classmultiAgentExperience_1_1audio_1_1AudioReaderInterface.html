Reads audio data from the underlying stream and copies it to the provided buffer.

Parameters
bufA buffer to copy the consumed data to. This buffer must be large enough to hold nBytes bytes
nBytesThe maximum number of bytes to copy
timeoutThe maximum time to wait for data. If this parameter is zero, there is no timeout and blocking reads will wait forever.
Returns
ssize_t The number of nBytes bytes copied if data was consumed.
Return values
0if the reader is disabled
-1if the data requested has been overwritten and is invalid
-3if no data becomes available before the specified timeout
-4if any parameter is invalid

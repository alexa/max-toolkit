Detailed Description

The Agent is an actor which handles input from the user to participate in the multi-agent experience.

The Agent is not initialized until it is registered. Be aware the MAX ActivityManagerInterface, DialogManagerInterface, and AgentTransferManagerInterface are not usable until onInitialized is called.

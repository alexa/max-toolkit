Called when the request is granted and the activity can start. The provided ActivityControllerInterface, ControlRegistryInterface, and ExperienceControllerInterface are valid only until onStop is called. Once onStop is called these interfaces will have no effect, and any registered Controls or started Experiences will be automatically cleaned up.

Parameters
activityController- Used to control the activity's lifecycle.
focus- The current focus of the activity, must be used as if it were given in onFocusStateChange.
controlRegistry- Used to register controls which impact this activity.
experienceControl- Used to start experiences for this activity.

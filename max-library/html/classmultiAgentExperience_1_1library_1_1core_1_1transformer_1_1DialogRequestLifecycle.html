Detailed Description

A DialogRequestLifecycle maintains a lifecycle for a registered DialogRequestInterface, ensuring its lifeycle is completed and no illegal transitions are made.

◆ beforeStop()

void multiAgentExperience::library::core::transformer::DialogRequestLifecycle::beforeStop ( )

Intermediate step between a started and stopped request. This must always be followed buy a call to stop. It is useful for taking the dialog out of a started state while some cleanup is being done before stop is called.

Note: An unstarted request will be denied instead.

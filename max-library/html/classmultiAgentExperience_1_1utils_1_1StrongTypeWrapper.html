Detailed Description

template<typename T, typename Tag>
class multiAgentExperience::utils::StrongTypeWrapper< T, Tag >

A Utility class for creating strongly typed aliases/typenames/wrappers of types.

e.g: using Length = StrongTypeWrapper<double, struct WidthTag> using Area = StrongTypeWrapper<double, struct AreaTag>

Length l = 5.0; Area a = l; //Compiler error

The "Tag" template parameter is used to differentiate wrappers over the same type. It need not be declared or used elsewhere.

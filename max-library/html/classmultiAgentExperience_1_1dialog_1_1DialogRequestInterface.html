Detailed Description

DialogRequestInterface contains callbacks for a dialog request's lifecycle. Each request is either denied or started, and any request which is started is also stopped. Following these lifecycle callbacks allows Agents to interact with any other multi-agent experiences.

A previously started dialog was stopped. This is called whether or not the dialog was stopped by the Agent which requested the dialog, or some other multi-agent experience interrupted it.
